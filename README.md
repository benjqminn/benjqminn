## 🧪 Featured Projects

### ☁️ Forensic Visibility in AWS - CloudTrail Analysis & Evidence Reconstruction

Semester-long graduate cloud forensics research project evaluating how effectively AWS-native logging can reconstruct simulated attacker activity.

Built a controlled AWS environment and achieved:

* **87.4% overall observability**
* **63.2% strict full reconstruction**
* **87 simulated attacker-style actions tested**

Key work included:

* Simulated IAM abuse, privilege escalation, infrastructure changes, and evasion techniques
* Collected evidence from CloudTrail, S3 logs, VPC Flow Logs, Systems Manager, and GuardDuty
* Built ground-truth timelines to compare real activity against recorded AWS evidence
* Implemented SHA-256 evidence verification and chain-of-custody-style documentation
* Adapted the final research into a public Medium article with redacted AWS details

Main takeaway:

**CloudTrail is excellent for AWS control-plane visibility, but it is not a complete forensic record without cross-source correlation.**
